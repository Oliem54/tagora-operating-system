# Template — Standard module

## Objectif

Fournir le modèle standard pour documenter l'adoption ou les spécificités d'un module TAGORA vis-à-vis de TOS.

## Portée

Documents dans `docs/04_adoption_modules/` et extensions par module.

## Principes

- Séparation standards communs vs spécificités module.
- Convergence progressive documentée.
- Règles métier du module restent dans l'application module.
