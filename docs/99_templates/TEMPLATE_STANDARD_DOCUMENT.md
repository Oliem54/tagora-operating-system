# Template — Document standard

## Objectif

Fournir le modèle standard pour créer un nouveau document de standard dans le TAGORA Operating System.

## Portée

Tous nouveaux fichiers dans `docs/02_standards/` ou standards par pilier.

## Principes

- Structure uniforme et prévisible.
- Pas de faux détail technique — brouillon évolutif acceptable.
- Généricité et durabilité.
- Statut explicite (Brouillon initial → Validé).

---

## Modèle

```markdown
# [Titre du standard]

## Objectif

[Pourquoi ce standard existe.]

## Portée

[À quoi s'applique ce standard.]

## Principes

- [Principe 1]
- [Principe 2]

## Règles

- [Règle 1]
- [Règle 2]

## Éléments à documenter plus tard

- [Item 1]
- [Item 2]

## Statut

**Brouillon initial**
```
