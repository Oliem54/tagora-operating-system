# TDS — TAGORA Design System

## Objectif

Définir le standard officiel de l'image de marque et de l'expérience utilisateur TAGORA, garantissant une identité visuelle commune à toutes les applications.

## Portée

Interfaces utilisateur de tous les modules TAGORA — web, mobile et futurs canaux.

## Principes

- **Signature TAGORA immédiate** — Time, Stock Premium, Voice, Mail IA doivent être reconnaissables instantanément.
- **Design Premium** — Interface épurée, professionnelle, moderne.
- **Accessibilité** — Utilisable par un public large (20 à 60 ans).
- **Cohérence** — Mêmes composants, couleurs, typographie, navigation.
- **Simplicité** — Informations critiques visibles ; détails en vues secondaires.

## Règles

- Toute interface TAGORA doit respecter la palette, typographie et composants officiels (à définir).
- Aucune surcharge visuelle ; priorité à la lisibilité.
- Responsive et accessible par défaut.
- Référencer `docs/09_design_system/` pour les standards détaillés.
