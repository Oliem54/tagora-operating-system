# TAF — TAGORA AI Framework

## Objectif

Définir le framework officiel pour l'intégration, la gouvernance et la validation des agents IA dans l'écosystème TAGORA.

## Portée

Agents IA, prompts, workflows IA, collaboration humain-IA et documentation IA — tous modules TAGORA.

## Principes

- **Limites d'intervention claires** — L'IA assiste ; elle ne remplace pas les décisions métier critiques.
- **Traçabilité** — Prompts, décisions et validations documentés.
- **Validation humaine** — Human in the Loop pour les actions sensibles.
- **Cohérence** — Standards de prompts et d'agents uniformes.
- **Sécurité** — Respect des règles de sécurité et de confidentialité.

## Règles

- Tout agent IA doit respecter les limites d'intervention définies dans TAF.
- Documenter l'architecture et le rôle de chaque agent.
- Valider les outputs IA selon les critères TAF avant action automatisée.
- Référencer `docs/07_ai/` pour les standards détaillés.
