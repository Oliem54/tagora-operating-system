# TAGORA Operating System (TOS)

## Objectif

Ce dépôt est la **Constitution méthodologique** de l'écosystème logiciel TAGORA. Il centralise les standards, l'architecture, les méthodes de travail, les bonnes pratiques et les règles applicables progressivement à tous les projets TAGORA.

**TOS n'est pas une application.** C'est la référence officielle pour la convergence méthodologique de l'écosystème.

## Portée

- TAGORA Time (projet pilote)
- TAGORA Stock Premium
- TAGORA Voice
- TAGORA Mail IA
- TAGORA Pulse
- TAGORA Platform
- Tous les futurs modules TAGORA
